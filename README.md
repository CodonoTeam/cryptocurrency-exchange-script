<details>

- **Backend:** PHP 8.1+ (Custom MVC framework)
- **Frontend:** Vue.js 3 with Tailwind CSS
- **Mobile App:** Ionic/Angular
- **Trading Engine:** Node.js (TypeScript)
- **Database:** MySQL, Redis, QuestDB (for time-series data)

</details>

<details>
<summary><b>How much does crypto exchange software cost?</b></summary>

<summary><b>Do I need to run blockchain nodes?</b></summary>

No, running your own nodes is optional. You can use third-party services like:

- **Blockgum** - For all EVM chains
- **CoinPayments** - For 2000+ coins
- **CryptoApis.io** - API-based integration

However, running your own nodes gives you more control, better security, and lower transaction fees.

</details>

<details>
<summary><b>Do I get the full source code?</b></summary>

Yes. Codono provides **100% unencrypted source code** — backend, frontend, mobile app, trading engine, and admin panel. You can modify, customize, and rebrand everything without restrictions.

</details>

<details>
<summary><b>How long does it take to launch?</b></summary>

Most operators go live in **3 to 5 days**. The automated installer sets up everything in about 30 minutes. Additional time is spent on branding, configuring coins, setting up KYC, and testing before public launch.

</details>
